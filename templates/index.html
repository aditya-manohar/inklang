Quick Reference

# Load dataset
dataset iris from "iris.csv"

# Train model
acc = train rf1 with iris 
  using RandomForest for 10 epochs

# Conditional logic
if acc > 0.9:
    print "Excellent model!"

# Repeat operations
repeat 3 times:
    print "Training..."

# Make predictions
predict rf1 with iris

# Save/load models
savemodel rf1 to "model.pkl"
loadmodel rf_saved from "model.pkl"

Note: The last column in your CSV/JSON is automatically used as the target variable.

All code execution happens server-side with proper sandboxing.

Trained models are automatically downloaded as .pkl files.

Code Examples

Basic Print and If

# Basic conditional with print
dataset iris from "iris.csv"
acc = train model1 with iris for 1 epochs
if acc >= 0.8:
    print "Model is good enough!"
print "Check complete."

Repeat and Print Loop

# Simple loop with print
repeat 5 times:
    print "Hello, Inklang!"
print "Loop done."

Combined Basic Syntax

# Combine print, if, and repeat
dataset iris from "iris.csv"
print "Starting check..."
repeat 2 times:
    acc = train model1 with iris for 1 epochs
    if acc > 0.85:
        print "Accuracy is high!"
    print "Iteration done."
print "All checks complete."

MNIST Classifier
