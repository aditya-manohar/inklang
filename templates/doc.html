Inklang Documentation

Learn the syntax of Inklang, a minimalist language for training and deploying machine learning models with elegant syntax.

Syntax Overview

Inklang is designed for simplicity, focusing on machine learning workflows. It supports a small set of statements for loading datasets, training models, making predictions, and controlling flow with conditionals and loops. Below is a detailed guide to each statement, including syntax, usage, and examples.

General Rules

  • Tokens: Identifiers (NAME: e.g., mnist, nn1), quoted strings (STRING: e.g., "iris.csv"), numbers (NUMBER: e.g., 10, 0.95), and comparison operators (CMP_OP: >, <, ==, >=, <=, !=).
  • Comments: Single-line comments start with # (e.g., # Example).
  • Indentation: Not required for parsing (whitespace is ignored), but recommended (4 spaces or 1 tab) for readability in if and repeat blocks.
  • Block Delimiters: Use : to start blocks in if and repeat statements.

Statements

Print Statement

Outputs a message to the console or website output.

Syntax:

print STRING

Example:

print "Hello, Inklang!"

Output: [inklang] Hello, Inklang!

Dataset Statement

Loads a dataset into memory from a file or built-in dataset.

Syntax:

dataset NAME from STRING

Example:

dataset iris from "Uploads\iris.csv"
dataset mnist from "mnist"

Notes: NAME is the dataset identifier; STRING is a file path or built-in dataset name.

Train Statement

Trains a model and stores its accuracy in the acc variable.

Syntax:

acc = train NAME with NAME ["using" NAME] for NUMBER epochs

Example:

acc = train nn1 with mnist using NeuralNetwork for 10 epochs
acc = train rf1 with iris for 100 epochs

Notes: using NAME is optional (defaults to NeuralNetwork); NUMBER must be an integer.

Predict Statement

Makes predictions using a trained model on a dataset.

Syntax:

predict NAME with NAME

Example:

predict nn1 with mnist

Save Model Statement

Saves a trained model to a file.

Syntax:

savemodel NAME to STRING

Example:

savemodel nn1 to "Uploads\mnist_model.pkl"

Load Model Statement

Loads a model from a file.

Syntax:

loadmodel NAME from STRING

Example:

loadmodel nn1 from "Uploads\mnist_model.pkl"

If Statement

Executes statements based on the acc variable’s value.

Syntax:

if acc CMP_OP NUMBER :
    statement+

Example:

if acc > 0.95:
    print "Excellent accuracy!"
    savemodel nn1 to "Uploads\model.pkl"

Notes: CMP_OP is >, <, ==, >=, <=, or !=. Indentation is recommended but not required.

Repeat Statement

Repeats a block of statements a fixed number of times.

Syntax:

repeat NUMBER times :
    statement+

Example:

repeat 3 times:
    print "Training iteration"
    acc = train nn1 with mnist for 5 epochs

Notes: NUMBER must be an integer. Indentation is recommended but not required.

Quick Reference

# Print a message
print "Hello, Inklang!"

# Load dataset
dataset NAME from STRING

# Train model
acc = train NAME with NAME [using NAME] for NUMBER epochs

# Make predictions
predict NAME with NAME

# Save/load models
savemodel NAME to STRING
loadmodel NAME from STRING

# Conditional logic
if acc CMP_OP NUMBER :
    statement+

# Repeat loop
repeat NUMBER times :
    statement+

# Comments
# This is a comment

Note: Indentation is optional but recommended (4 spaces or 1 tab) for readability in if and repeat blocks.
